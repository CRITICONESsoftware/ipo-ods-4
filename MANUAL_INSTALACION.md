# Manual de Usuario: Proyecto ODS-4 (Versión Offline)

Este proyecto ha sido optimizado para su ejecución en entornos sin conexión a internet o para una revisión rápida. Siga los pasos a continuación para visualizar la aplicación:

## 1. Instrucciones de Ejecución

Para ver el proyecto correctamente, solo necesita seguir estos dos pasos:

1.  **Descomprimir el archivo**: Localice el archivo `web.rar` (o `web.zip`) y extraiga su contenido en una carpeta de su elección.
2.  **Abrir la aplicación**: Entre en la carpeta extraída y haga doble clic en el archivo `index.html`.

La aplicación se abrirá en su navegador predeterminado y todas las funcionalidades (incluyendo el reproductor de vídeo con subtítulos y el cuestionario) funcionarán de forma local.

---

## 2. Notas Importantes

- **Reproductor de Vídeo**: El reproductor detectará automáticamente que se encuentra en modo offline y utilizará el archivo de vídeo local y los subtítulos integrados.
- **Navegador Recomendado**: Se recomienda utilizar una versión reciente de Google Chrome, Microsoft Edge o Mozilla Firefox para asegurar la mejor experiencia visual.
