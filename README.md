# Proyecto ODS-4 (ipo-ods-4)

Este es el repositorio oficial del proyecto **ipo-ods-4**, desarrollado con Next.js, React 19 y Tailwind CSS 4.

## Guía de Instalación y Desarrollo

Para configurar el entorno de desarrollo y comenzar a trabajar en el proyecto, consulta el manual detallado a continuación.

---

### 1. Requisitos Previos

Asegúrate de tener instaladas las siguientes herramientas:

- **Node.js**: v18.0.0+ (LTS recomendada).
- **NPM**: Incluido con Node.js.
- **Git**: Para control de versiones.

### 2. Configuración Inicial

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/tu-usuario/ipo-ods-4.git
   cd ipo-ods-4
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

### 3. Scripts Disponibles

| Comando | Acción |
| :--- | :--- |
| `npm run dev` | Inicia el servidor de desarrollo en `localhost:3000`. |
| `npm run build` | Genera el build de producción. |
| `npm run start` | Arranca la aplicación en modo producción. |
| `npm run lint` | Analiza el código en busca de errores de estilo. |

### 4. Tecnologías Principales

- **Framework**: Next.js 15+ (App Router)
- **UI**: React 19, Radix UI, Lucide Icons.
- **Estilos**: Tailwind CSS 4.0.
- **Componentes**: shadcn/ui.

### 5. Estructura de Directorios

- `/app`: Rutas y lógica de servidor.
- `/components`: Elementos visuales (UI y piezas específicas de ODS).
- `/lib` y `/hooks`: Utilidades y lógica de estado compartida.
- `/public`: Archivos estáticos.
